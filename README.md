# Signal Notes

Short, honest write-ups compiled from things worth saving: interesting posts,
tools, and updates from projects in progress. Each piece credits its original
source and adds context rather than just reposting.

**Read them here:** https://flipthiscrypto.github.io/Obsidian-Signal-Notes/

This site is generated automatically. Every article started as a bookmark or a
commit, was written up in one voice, and published with an original cover and a
simple explanatory image. The goal is to turn daily curiosity into useful public
notes.
